# SwdroidLib

android项目常用工具集

#### gradle引用 ####
1. `	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}`
2. `dependencies {
	        compile 'com.github.xusx1024:SwdroidLib:1.0.0'
	}`
  
#### maven引用 ####
1. `<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>`
2. `<dependency>
	    <groupId>com.github.xusx1024</groupId>
	    <artifactId>SwdroidLib</artifactId>
	    <version>1.0.0</version>
	</dependency>`
